A clean and interactive Palindrome Checker built with HTML, Tailwind CSS, and Vanilla JavaScript.
This tool analyzes any word or phrase, ignores spaces and punctuation, and checks whether it reads the same forwards and backwards. Features include dark mode, input cleaning, reversal preview, and smooth UI feedback.
This project highlights skills in DOM manipulation, UI development, logic implementation, and theme management.
